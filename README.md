## How I build

1. 先确定**搜索意图**和可服务的**原子能力**  
2. 做出**能被索引**的最小上线版本（页面可渲染、加载快、响应快）  
3. 给出**清晰的使用路径**与**分享钩子**（Embed/分享卡片/一键复制）  
4. 用数据回收：查询词、点击路径、留存与转化 → 迭代

> 原则：做能“马上被用”的东西，其余都靠后。
